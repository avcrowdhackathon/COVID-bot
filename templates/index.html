Features

By understanding simple keywords and their meaning, it gives answers to simple and complex questions, myths, facts, data about COVID-19, identifies different personas, and provides tailored information to each specific user.

Purpose

COVID-bot provides meaningful and accurate answers to the user queries, regarding COVID-19 precautions, preventions, and the current situation per country. The aim is to be used as the main dissemination prevention tool 24/7, replacing the hotlines of each country.

Who Replies?

A chatbot is a digital assistant that simulates human chat conversation through artificial intelligence (AI) and machine learning (ML). This is an English Beta Version. Interacting with the bot, helps it become smarter.


Add COVID-bot to your website

Would you like your website visitors to know that you care for them ? If yes, then add COVID-bot pop-up to your website and help them get informed about COVID-19. You can easily embed COVID-bot to your own website by following the steps explained on our GitHub Repository.
